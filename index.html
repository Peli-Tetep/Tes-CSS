2. Internal CSS

Diletakan di <head>.

Contoh:

<style>
 body{background-color:yellow;}
 h1{color:red;}
 p{color:blue;}
</style>

Hasilnya:

Header merah

Paragraf biru.

3. External CSS

Diletakan di file khusus CSS (.css), dan diletakan pada <head>.

Contoh:

<link rel="stylesheet" href="style.css">

dengan isi file css sebagai berikut :
 body{background-color: red;}
 h1{color: blue;}
 p{color: yellow;}

Hasilnya:

Header merah

4. Universal Selectors

Contoh 1:

<h1>Header</h1>
<p>Bakal ngubah semua yang ada disini</p>

dengan kode stylenya:
* {
 color: blueviolet;
 text-align: center;
}

Hasilnya:

Header

Bakal ngubah semua yang ada disini

5. Groupping Selectors

Contoh 1:

<h1>Header</h1>
<p>Bakal ngubah semua yang ada disini</p>

dengan kode stylenya:
*h1, p {
 color: red;
}

Hasilnya:

Header

Bakal ngubah semua yang ada disini
